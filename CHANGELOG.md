### Added

- AI chat integration with Ollama
- File management (save, load, auto-save)
- System tray integration
- Tutorial mind map for new users
- Summary nodes for conversation consolidation

- Create and edit mind map nodes
- Connect nodes with edges
- AI-powered chat using local Ollama models
- Automatic and manual file saving
- Mind map persistence across sessions
- Customizable node types and styling
